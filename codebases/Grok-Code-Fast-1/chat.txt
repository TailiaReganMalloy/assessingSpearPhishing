I've created a complete Node.js example application for your students that demonstrates secure user authentication and messaging functionality. Here's what I've built:

Project Structure
server.js: Main application with secure authentication, password hashing, and message handling
views/: EJS templates for login, registration, inbox, and compose pages
public/styles.css: Clean, responsive styling
package.json: Dependencies including bcrypt for password hashing, express-session for authentication, and SQLite for data storage
README.md: Comprehensive documentation explaining the features and security practices
Key Security Features Demonstrated
Password Hashing: Uses bcrypt to hash passwords before storage (never store plain text passwords)
Session Management: Secure session-based authentication
SQL Injection Protection: Parameterized queries prevent SQL injection attacks
Access Control: Users can only see their own messages
Input Validation: Basic validation on forms
Features
User registration with secure password storage
Login/logout functionality
Message composition and sending between users
Inbox view showing received messages
SQLite database for persistence
