‚ùì FAQ

Q: How long does setup take?

A: About 2-5 minutes. Just run npm install and npm start. See QUICKSTART.md.

Q: What if I get an error?

A: Check QUICKSTART.md "Troubleshooting" section. Most issues are missing Node.js or port conflicts.

Q: Where is the database?

A: SQLite database is created as app.db in the main folder. Auto-created on first run.

Q: Can I modify the code?

A: Yes! That's encouraged. Use npm run dev for auto-reload when you make changes.

Q: Is this production-ready?

A: No - this is educational. See README.md "Production" section for requirements before deploying.

Q: What should I submit?

A: Complete ASSIGNMENT.md with your work, screenshots, and any code modifications.
