# Secure Login and Messaging Demo

This is a demonstration web application built with Node.js and Express.js to teach secure login mechanisms, safe password storage, and inter-user messaging.

## Features
- User registration with password hashing (bcrypt)
- Secure login with JWT authentication
- Dashboard for sending and viewing messages
- SQLite database for storage

## Setup
1. Install dependencies: `npm install`
2. Run the server: `node app.js`
3. Access at http://localhost:3000

## Educational Notes
- Passwords are hashed using bcrypt.
- Sessions managed with JWT.
- See code comments for more details.
