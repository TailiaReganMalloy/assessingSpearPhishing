# BlueMind v5

A secure login and messaging application demonstrating safe password storage and user authentication.

## Features

- Secure login with bcrypt password hashing
- Session management using express-session
- Inter-user messaging (demo)

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the server:
   ```bash
   npm start
   ```

3. Visit `http://localhost:3000` in your browser.

## Dependencies

- Express
- Mongoose
- bcryptjs
- EJS
- express-session
